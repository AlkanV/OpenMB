# AMOFGameEngine

## Introduction:
A game engine based on AdvancedMogreFramework

## Feature:
* Ogg Support(done)  
* AppState Management(done)  
* Character Controller(done)  
* Network(further)  
* Physx(doing)  
* GUI System(done, current:mogrebites) 
* AI(doing)
