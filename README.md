# AMOFGameEngine

## Introduction:
A game engine based on AdvancedMogreFramework

## Feature:
* Ogg Support  
* AppState Management  
* Character Controller  
* Network  
* Physx  
* GUI System 
* AI  

## Based on:
* Mogre(Graphic Render Engine;LGPL)  
* NAudio(Sound Library;Ms-PL)  
* NVorbis(Library for Ogg Vorbis File;Ms-PL)  
* MogreBites(GUI System;GPLv2)  
* Prolog.NET(AI;Ms-PL)
